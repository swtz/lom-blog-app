Aula: 119. Componente PostImageCover com Link + Image do Next

Dica para definir as 'props' de um componente de modo que sejam iguais
a algum componente React ou Next já existente:

type PostCoverImageProps = {
  imageProps: React.ComponentProps<typeof Image>;
  linkProps: React.ComponentProps<typeof Link>;
};

Assim, os componentes 'Image' e 'Link' podem receber todas os propriedades
nativas mais personalizadas que o dev quiser criar.

Em contrapartida, o coach-Luiz ressaltou que essa prática pode tornar mais árduo
o desenvolvimento de testes pela quantidade a propriedades que um Componente React/Next
nativo possui.
