Aula: 112. Loading.tsx e Suspense para exibir fallback quando algo estiver carregando

Para usar a funcionalidade de 'loading' no Next, é possível usar o componente 'Suspense'.

Estrutura:

<h1>Olá, mundo!</h1
<Suspense fallback={<SpinLoader />}>
  <PostsList />
</Suspense>
<footer>Olá, footer!</footer>

O componente 'PostsList' é assíncrono; consequentemente, possui métodos assíncronos.
O ponto é: para a funcionalidade de 'loading' funcionar corretamente, todo o trecho
de código assíncrono precisa estar envolvido no componente 'Suspense'.

O restante dos elementos JSX acima — h1 e footer — são renderizados imediatamente.
Enquanto isso — que é o período chamado 'fallback' — é exibido o componente criado
anteriormente 'SpinLoader'. Assim, o Next permite gerenciar a exibição do componente
de 'loading' com base no status de carregamento do que está envolto em 'Suspense'.

Novamente, é preciso ficar atento à maneira como os componentes da aplicação estão
estruturados.
