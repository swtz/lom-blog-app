Aula: 113. Container geral do site + tema claro e escuro no Tailwind CSS

Ao trabalhar com TW, quando se define uma cor no elemento Pai, os elementos Filhos herdam
essa cor. Por isso é preciso sobrescrever essa cor nos elementos Filhos (se desejar).

A aula mostra como utilizar as classes do TW para trocar o 'theme' da aplicação com base
no 'theme' do sistema operacional.

Entretanto, caso o dev queira fazer isso manualmente, basta definir no arquivo 'globals.css':

@custom-variant dark (&:where(.dark, .dark *));

Assim, quando presente na tag 'html' a classe 'dark', o 'theme' do app muda.

Obs.: é possível escolher somente um dos dois modos (automático ou manual).
