Dica CSS:

Quando se aplica um display 'flex' no elemento Pai, os elementos Filhos se comportam
como display 'block'. Isso é importante, pois algumas propriedades só se aplicam
a elementos que possuem display 'block' (tal como as bordas arrendondadas de um elemento
quadrado).

Por exemplo, se uma imagem está dentro de um elemento 'a', não será aplicado esse efeito
das bordas, pois, por padrão, elementos 'a' são 'inline'. Assim, quando se aplica o
display 'flex' no elemento Pai desses links, o efeito das bordas é aplicado.
