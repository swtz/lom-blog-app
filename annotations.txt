Seção 3: Next.js 15, React 19 e Tailwind CSS v4: Introdução ao Desenvolvimento Moderno

Aula: 114. Componente Container + Dicas do Tailwind CSS

O TW CSS em si é mobile first, ou seja, ao criar os estilos, é recomendável
começar com tamanhos de tela menores e ir crescendo as medidas ao longo do desenvolvimento.

Uma das vantagens de criar 'Containers' de estilo é que o ponto de alteração é reduzido
a um componente. Caso seja feita uma mudança, ela é refletida para todos os componentes da
aplicação que usam esse Componente de estilo.
