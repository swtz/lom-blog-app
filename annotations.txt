Aula: 183. ImageUploader - validando o tamanho do arquivo no front-end

A função 'handleChange', que é disparada quando o usuário tenta fazer upload de um arquivo
possui alguma checagens importantes para validação de imagens. É importante imprimir esse
arquivo no console, pois a imagem possui algumas propriedades que requerem atenção, pois
alguma dessas propriedades podem ser maliciosamente manipuladas por alguém que quer enviar
algo malicioso, porém se passando por uma imagem.

Uma dica do coach-Luiz é usar uma lib externa tal como a 'sharp'
https://sharp.pixelplumbing.com/
pois ela pode ajudar a verificar se o arquivo enviado é uma imagem de fato.

Sempre que o usuário tem a possibilidade de enviar algo para o servidor, é preciso prestar
muita atenção nas consequências de NÃO GARANTIR que, se o quê de fato foi enviado, é aquilo
que apresenta ser.
