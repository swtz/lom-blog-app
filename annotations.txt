Dica CSS:

Quando um elemento pai possui a css rule 'flex flex-col', por padrão a css rule 'items-stretch'
é aplicada, assim os elementos filhos esticam o máximo possível para ocupar 100% do espaço disponível.

Caso o programador queira modificar esse comportamento em um único elemento filho, é possível
usar a css rule 'self-[value]' nesse elemento filho.
