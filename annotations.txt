Aula: 123. Exibindo a data formatada em nossos componentes

<time
  className='text-slate-600 block text-sm/tight'
  dateTime={post.createdAt}
  title={formatRelativeDate(post.createdAt)}
>
  {formatDatetime(post.createdAt)}
</time>

O atributo 'dateTime' é reconhecido pelo browser para indicar que a tag 'time' possui
uma data de fato (tanto é que seu valor não é formatado).

Para renomear o nome de uma variável, é possível selecioná-la e apertar a tecla 'F12',
assim, ao renomear, o VS Code faz as alterações em todos os arquivos que usam essa variável.
