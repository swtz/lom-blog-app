Aula: 147. Conexão Drizzle ORM + migrations para gerenciar as tabelas da base de dados

Decidi seguir a aula à risca (uso do better-sqlite3 ao invés do sqlite3):

npm i drizzle-orm better-sqlite3
npm i -D drizzle-kit @types/better-sqlite3

Com o arquivo de configuração pronto e a conexão feita (arquivo 'src/db/drizzle/index.ts'),
é possível testar se o banco de dados está funcionando. Basta informa no terminal:

npx drizzle-kit generate | são geradas as migrações da base de dados, ou seja, os comandos
SQL a serem executados

npx drizzle-kit migrate | os comandos gerados são aplicados.

Ao informar esse último comando, é preciso ter cuidado, pois ele altera o banco de dados
definitivamente.

Nota: os comandos usados para teste foram instruídos em um novo arquivo dentro do diretório
que contém a conexão.
